# Product Positioning

## 한 줄 정의

ModelMate는 비전공자를 위한 CSV 기반 AutoML 데모 서비스다.

## 비교 기준

| 서비스 | 강점 | ModelMate와 차이 |
| --- | --- | --- |
| Google Vertex AI | 기업용 AutoML, 강력한 클라우드 기능 | 진입 장벽이 높고 설정이 복잡하다 |
| DataRobot | 모델 관리, MLOps, 대규모 운영 | 졸업프로젝트 범위를 넘어서는 기업용 기능이 많다 |
| Obviously AI | 비전공자 친화적 예측 분석 | ModelMate가 지향하는 사용성 방향과 가장 가깝다 |
| H2O Driverless AI | 강력한 AutoML과 설명 기능 | 전문 사용자와 기업 환경에 더 가깝다 |

## 우리 프로젝트의 포지션

ModelMate는 대형 AutoML 서비스처럼 모든 운영 기능을 제공하는 것이 목표가 아니다.

목표는 다음과 같다.

- CSV 파일만으로 시작할 수 있다.
- 모델 선택 과정을 자동화한다.
- 결과를 비전공자도 이해할 수 있게 요약한다.
- 예측 근거를 함께 보여준다.

## 발표에서 강조할 차별점

- 복잡한 AI 용어보다 사용 흐름을 먼저 보여준다.
- 모델 성능뿐 아니라 예측 근거도 제공한다.
- 기능을 5단계 핵심 흐름으로 단순화했다.
- 예측과 API 공유는 선택 확장 기능으로 제공한다.
