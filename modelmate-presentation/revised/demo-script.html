다음: CSV 업로드 후 자동 판단

4CSV 업로드

약 55초
핵심 보강

범용성 포인트: 데이터셋 종류에 따라 타겟과 입력 변수가 달라짐.

CSV를 업로드하면 ModelMate가 데이터셋에 맞게 자동으로 읽고 정리합니다.

예를 들어 의료나 건강 데이터라면 진단 관련 예측 문제로 정리할 수 있고, 공공교통 데이터라면 이용자 통계나 수요 예측 문제로 연결할 수 있습니다.

업로드 후에는 행 수, 컬럼 수, 결측값, 데이터 종류를 요약하고, 사용할 정보와 제외할 정보를 나누어 보여줍니다.

여기서 중요한 점은 고정된 샘플만 처리하는 것이 아니라, 데이터셋 종류에 따라 맞힐 값과 입력 변수가 달라진다는 점입니다.

다음은 AI 한 번에 실행 기능입니다.

이 기능은 사용자가 직접 모델을 고르지 않아도, AI 분석 코치가 데이터 상태 확인부터 모델 비교, 개선 판단, 이유 분석까지 한 번에 진행하는 흐름입니다.

이 기능의 목적은 분석 과정이 블랙박스처럼 보이지 않도록 하는 것입니다.
