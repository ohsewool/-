Concept

CSV 기반 예측 실험 자동화

ModelMate는 단순히 모델 점수를 보여주는 도구가 아니라, CSV를 예측 가능한 문제로 정리하고 모델 비교, 설명, 재사용까지 연결하는 웹 서비스입니다.

1

CSV 업로드

파일 구조와 데이터 품질 확인

2

분석 설정

맞힐 값과 제외 컬럼 정리

3

모델 비교

교차검증 기반 성능 비교

4

결과 설명

성능과 변수 중요도 해석

5

결과 재사용

예측, API, 기록으로 연결

Implementation

React/Vite · FastAPI 기반 웹 서비스 구현

Frontend

업로드, 모델 선택, 결과 요약, XAI, AI 분석 코치 화면을 구성했습니다.

Backend

CSV 처리, 데이터 검증, 타겟 추천, 모델 학습/비교 API를 구현했습니다.

AutoML

분류/회귀를 판별하고 여러 모델을 교차검증으로 비교합니다.

Experiment Log

로그인 사용자별 분석 기록과 결과 재확인 흐름을 제공합니다.

Service Screens

핵심 서비스 화면

발표에서는 전체 스크린샷을 모두 보여주기보다, 사용자가 실제로 거치는 핵심 흐름만 보여줍니다.

CSV 업로드

파일을 올리면 데이터 구조와 예측 가능성을 먼저 확인합니다.

모델 비교

여러 모델을 같은 기준으로 비교하고 적합한 모델을 선택합니다.

결과 요약

성능, 선택 모델, 중요한 정보, 해석 결과를 한 화면에서 확인합니다.

Agentic Flow

분석 문제 정리와 다음 행동 추천

AI 분석 코치는 업로드된 CSV를 보고 데이터 분야, 예측 목적, 타겟 후보, 제외 컬럼, 모델 비교 진행 가능 여부를 사용자에게 설명합니다.

데이터 분야 판단

건강, 제조 품질, 공공교통, 시설 안전, 고객/매출 등 컬럼 패턴을 기반으로 도메인을 추정합니다.

타겟/제외 컬럼 판단

모델이 맞혀야 할 값과 예측을 방해하거나 누수를 만들 수 있는 컬럼을 분리합니다.

다음 단계 안내

바로 모델 비교를 진행할지, 데이터를 다시 확인해야 할지 명확한 행동으로 연결합니다.

Validation

대표 데이터 출처 기반 검증 결과

UCI 공개 데이터셋

당뇨, 품질, 분류/회귀 예제처럼 머신러닝 학습에 자주 쓰이는 정형 데이터로 기본 AutoML 흐름을 확인했습니다.

데이콘 데이터셋

수면·건강 라이프로그처럼 컬럼명이 축약된 데이터에서 타겟 의미를 사람이 이해하는 말로 바꾸는지 확인했습니다.

공공기관 데이터

공공자전거, 시설 관리 등 한글 컬럼과 행정 데이터가 포함된 CSV에서 도메인 추론과 인코딩 처리를 확인했습니다.

비정상 파일

잡담성 텍스트, 너무 작은 파일, 예측 대상이 불명확한 파일을 안내 메시지로 처리하도록 개선했습니다.

Future Work

범용 AutoML 서비스로의 확장 방향

데이터 품질 진단

모델 관리

저장 모델, 버전 기록, 재학습, 새 데이터 예측 흐름을 강화합니다.

SaaS 기능

사용자별 작업 공간, 공유 링크, API 키, 관리자 화면을 고도화합니다.
