# ModelMate Beta Feedback Message Draft

Use or adapt this short message when inviting a beta tester.

## English

Hi, I am preparing a beta test for ModelMate, a guided CSV predictive analysis
MVP.

ModelMate helps turn a CSV into a prediction target, model comparison,
explanation, grounded report, and reusable prediction API-style output.

Could you try it for about 5-10 minutes?

You can either upload your own safe CSV or use one of the sample datasets on the
upload screen. Please do not upload sensitive personal information, payment
data, medical data, passwords, API keys, or private customer records.

I would especially like feedback on:

- whether it is clear what CSV to upload;
- whether the target recommendation makes sense;
- whether warnings and limitations help you trust the result;
- whether the report is useful;
- whether the prediction API reuse idea makes sense;
- what is confusing;
- what would make you pay for this.

Feedback link: `<add Google Form, Typeform, or GitHub issue link>`

Thank you.
