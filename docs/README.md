## Naming Rules

- Product name: ModelMate
- Positioning: guided AutoML SaaS MVP 또는 CSV 예측 분석 워크스페이스
- Korean-first docs and UI copy
- Allowed technical terms: CSV, API, AutoML, SaaS, ROC-AUC, F1, SHAP, JSON,
  token, endpoint, cURL, Python, HTTP, URL, request ID, error ID, Railway,
  FastAPI, React, scikit-learn

## Claim Rules

Do not claim that ModelMate is:

- a fully mature enterprise SaaS
- a replacement for large cloud/enterprise AutoML platforms
- a system that guarantees prediction accuracy
- a completed compliance/security platform
- an active paid billing product
- a full MLOps platform

Acceptable claim:

- guided CSV predictive analysis SaaS MVP
- Agentic AutoML upgrade direction
- grounded report and reusable prediction API foundation
- beta/demo-ready portfolio project
