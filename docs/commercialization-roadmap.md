# Commercialization Roadmap

ModelMate is positioned as a guided CSV predictive analysis SaaS MVP. It is not
an enterprise AutoML replacement.

## Current Foundation

- Auth-lite bearer-token sessions with logout revocation.
- Guest demo mode for sample-data evaluation.
- User-owned project foundation with MVP access control.
- Agentic AutoML trace, trust panel, evidence summary, grounded report, and
  prediction API documentation.
- Release QA, auth smoke, ownership smoke, and product smoke checks.

## Near-Term PRs

1. Persistent project history and rerun ownership.
2. Report-id based save/export ownership.
3. Prediction API token metadata hardening.
4. Small usage guardrail polish.
5. Beta feedback loop and demo reliability improvements.

## Out of Scope for Now

- payment or billing;
- enterprise SSO;
- full RBAC;
- team workspace;
- full async job queue;
- feature store or full MLOps platform;
- automatic retraining loop.

## Honest Positioning

Use: "guided CSV predictive analysis SaaS MVP".

Avoid: "enterprise-grade authentication", "complete multi-tenant security",
"SOC2-ready", "DataRobot alternative", or "fully autonomous AI analyst".
