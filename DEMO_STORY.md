# ModelMate Demo Story

## 핵심 메시지

ModelMate는 데이터 분석을 모르는 사람도 CSV 파일만 넣으면 예측 모델을 만들고, 결과와 예측 근거까지 확인할 수 있게 돕는 AutoML 서비스다.

## 11월 발표 기준 데모 목표

- 30초 안에 "CSV를 넣으면 예측 AI를 만들어주는 서비스"라고 이해되어야 한다.
- 5분 안에 업로드, 분석, 모델 비교, 결과 요약, 이유 보기까지 자연스럽게 이어져야 한다.
- 7분 이상 발표할 경우 새 데이터 예측과 모델 공유 API를 확장 기능으로 보여준다.
- 발표 중 외부 AI 토큰이 부족해도 데모 모드와 저장 결과로 시연이 이어져야 한다.

## 추천 발표 흐름

1. 문제 제기
   - 데이터는 있지만 모델을 직접 만들기 어려운 사람이 많다.
   - 전처리, 모델 선택, 성능 해석, 예측 이유 설명은 비전공자에게 특히 어렵다.

2. 데이터 넣기
   - CSV 파일을 업로드한다.
   - AI가 예측할 값과 제외할 정보를 정리한다.

3. 자동 분석
   - 여러 모델을 자동으로 비교한다.
   - 필요하면 성능 개선을 시도한다.

4. 모델 고르기
   - 가장 성능이 좋은 모델을 확인한다.
   - 점수는 발표에서 자세히 설명하지 않고 "모델 비교 결과"로 짧게 말한다.

5. 결과 요약
   - 선택된 모델, 예측 유형, 사용한 정보 수를 보여준다.
   - 이 화면이 발표의 메인 결과 화면이다.

6. 이유 보기
   - AI가 어떤 정보를 중요하게 봤는지 보여준다.
   - "성능만 보여주는 것이 아니라 왜 그런 결과가 나왔는지 설명한다"가 핵심이다.

## 선택 확장 기능

- 새 데이터 예측: 학습된 모델에 새 값을 넣어 결과를 확인한다.
- API 공유: 모델을 다른 서비스에서도 쓸 수 있게 준비한다.
- 작업공간: 데이터셋, 실험 기록, 저장 모델을 이어서 확인한다.
- 실험 비교: 여러 번의 모델 비교 결과 중 어떤 결과가 나았는지 비교한다.

이 두 기능은 발표 핵심 흐름이 아니라 추가 기능으로 소개한다.

## 대표 데이터셋 후보

- Pima diabetes: 의료/건강 예측 예시로 이해하기 쉽다.
- 제조/설비 품질 데이터: 고장/불량 분류 예시로 프로젝트 문제 정의와 잘 맞는다.
- 공공자전거/공공시설 데이터: 공공기관 데이터 대응력을 보여주기 좋다.

발표 메인 데모는 하나만 고정하고, 나머지는 "다른 데이터셋도 처리 가능"을 보여주는 보조 사례로 둔다.
